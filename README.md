# Details
A Python tool that automatically extracts tables from websites using Selenium for browser automation and Google Gemini Vision AI for intelligent table extraction.

- Process multiple URLs in one run to reuse driver
- Increase timeouts for slow websites
- Reduce screenshot size for faster processing
